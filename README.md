# 🔗 Shorty – URL Shortener

Shorty is a modern, production-ready URL shortening application built using **Next.js (App Router)** and **TypeScript**. It provides a clean, elegant user experience with a beautiful frosted glass (glassmorphism) interface while delivering fast and secure URL shortening.

Designed with simplicity and performance in mind, Shorty offers an intuitive landing page, responsive design, and smooth interactions suitable for both desktop and mobile devices.

---

## ✨ Features

- ⚡ Lightning-fast URL shortening and redirection
- 🎨 Modern Glassmorphism (Frosted Glass) UI
- 🌈 Beautiful gradient background with soft glow effects
- 📱 Fully responsive design
- 🚀 Built with Next.js App Router
- 🔒 URL validation for secure redirects
- 💻 TypeScript support for better maintainability
- 🎯 Smooth modal interactions with automatic input focus
- 🌐 Optimized for performance

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/vivekjoshi2006/URL-Shortener.git
```

### 2. Navigate to the Project

```bash
cd url-shortener
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

### 5. Open in Your Browser

Visit:

```text
http://localhost:3000
```

---

## 📦 Build for Production

Create an optimized production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
